## Three-Step Call Sequence (Formalized)

```
Day 1 PM — Cold call (anticipation + brief intro)
Day 2 AM — DM with preview link ("Eto na po yung sabi ko")
Day 2 PM — Call with DM context ("Na-check n'yo na po ba?")
```

**Key: Voice connection makes DM land harder.** Never DM before first call.

---

## Staff / Non-Owner Branch (Mandatory in Every Script)

Every call script MUST include a section for when staff/receptionist answers:

1. First confirm: "Kayo po ba ang owner or manager?"
2. If NO → ask for specific callback time, not vague "balik ka na lang"
3. Don't pitch to staff — they can't decide
4. Get a specific time, send brief FB message with reference
5. On follow-up if same staff: "Kumusta! Naipakita n'yo ba sa manager?" — build rapport

**Why:** Better Barbers had receptionist answer twice. Same pattern will repeat.
