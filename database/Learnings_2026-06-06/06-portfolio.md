# Portfolio Design & Strategy

> Source: `memory/portfolio-design-decisions.md`, `memory/pre-build-portfolio-strategy.md`, `dev-handoff.md`

## Purpose

Credibility link to text after cold calls. Not SEO/content marketing. The portfolio does the selling — operator age is invisible behind professional design.

## Design Decisions (Approved 2026-06-03)

**Layout:** Single page, Show-First Flow

**Section order:**
1. **Hero** — Outcome-First headline ("Your Business, Online Today") not name-first
2. **Example Sites** — 3 screenshot cards (not device mockups or carousel)
3. **About** — Problem framing ("Most local businesses have no web presence") not personal bio
4. **Pricing** — ₱18k generic / ₱35k+ custom
5. **Contact** — Phone + email
6. **Footer**

**Aesthetic:** Glassmorphism + neumorphism (same design language as JCA 1221 Holdings at https://jca1221.netlify.app/)

**Stack:** React + Tailwind, Vite, Netlify deploy (Design OS pattern)
