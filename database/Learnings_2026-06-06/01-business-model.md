# Business Model

> Source: `WebDropshipping_Learnings.md`, `summer-web-business-learnings.md`, `memory/pricing-tiers.md`

## Core Hook

Free 24-hour live mockup delivered as GitHub Pages URL. Client opens it on their phone like a real site before committing any money. Stronger than screenshot or Figma because it's live and browsable.

## Pricing (PHP only — never quote USD)

| Tier | Price | Tool | Time | Details |
|------|-------|------|------|----------|
| **Generic** | ₱18,000 | Pre-built HTML template | <3 hrs delivery | 2 revisions, 30-day support |
| **Custom** | ₱35,000+ | Design OS → React+Tailwind | Open-ended | Unique design, 2 revisions, 30-day support |
| **Maintenance** | ₱2,500/mo | — | — | After 30-day support window |

**Payment:** 50% deposit on mockup approval, 50% on launch. Never 100% at the end — protects against ghosts.

**Why ₱18k:** PH agencies charge ₱7k–₱35k for basic sites, ₱50k+ for premium. ₱18k sits above template-reseller floor and the ₱15k blind spot, while still below agency pricing. Free preview hook justifies the gap.

## Workflow

**Generic tier:**
1. Identify business type on call
2. Clone pre-built template for that type
3. Swap: name, colors, logo, photos, copy, contact info
4. Push to GitHub Pages → send URL

**Custom tier:** Design OS → React+Tailwind. Bid per project.

## The September Problem

Operator returns to school in ~10 weeks. Every launched site = support liability.

**Contract must include:** 30 days post-launch support. After that: maintenance retainer or handoff training.

## Upsell Opportunities (activate after workflow smooth, ~Week 4)

- Google Business Profile setup (+₱1,000–₱1,500)
- Basic SEO (+₱1,500–₱2,000)
- Monthly maintenance retainer (₱2,500/mo)
- Social media graphics (+₱1,500–₱2,000)

## Hosting & Domain

- **Hosting:** GitHub Pages (free). Custom domain via CNAME.
- **Domain:** Client buys themselves from Namecheap/Google Domains (~$12-15/yr). Guide them through it.
- **SSL:** Free via GitHub Pages + custom domain.
- **Brief clients** that 24h preview URL is temporary GitHub link, not final domain.
